# BriefForge

BriefForge is a browser-only tool that turns messy AI, software, SaaS, or automation notes into a structured project brief.

## What It Generates

- Project summary
- Real problem
- Existing materials
- Missing information
- Smallest useful version
- Deliverable recommendation
- Workflow milestones
- Risks and non-goals
- Clarifying questions
- Next step

## Use Case

This is a free lead-in tool for the paid AI Brief Cleanup service.

Suggested funnel:

1. Visitor uses BriefForge.
2. Visitor copies or downloads the generated Markdown.
3. Visitor sends the draft through Upwork, Fiverr, or direct contact.
4. Paid service turns the draft into a more useful handoff brief.

## Local Use

Open `index.html` in a browser.
